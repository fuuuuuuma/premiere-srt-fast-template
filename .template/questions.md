# 壁打ち質問リスト

以下の質問を **1問ずつ** ユーザーに聞いてください。
回答がすべて揃ってからスキルを生成します。

## チャンネル設定

**Q1. チャンネル名は何ですか？**
- テロップの自己紹介文などで使う（例:「皆さんこんにちは〇〇の△△です」の〇〇部分）
- 無ければ空でも進められる

**Q2. テロップの目標文字数はどのくらいにしますか？**
- デフォルト: 平均14字前後・25字超は全体の1%未満
- 迷ったらデフォルトのまま進めてよい

**Q3. このチャンネルでよく出る固有名詞・製品名・人名で、Whisperが誤変換しそうなものはありますか？**
- 分かる範囲でよい（例:「クロードコード」→「ClaudeCode」のような言い間違い→正規表記のペア）
- 無ければ空でよい。生成のたびに気づいたものを追記していける運用にする

**Q4. 半角スペースの使い方に強いこだわりはありますか？**
- デフォルト: 話者のポーズ（一拍以上の間）と日本語/英数字の境界にのみ挿入
- 無ければデフォルトのまま進めてよい

---

## 転写エンジン

**Q5. Mac（Apple Silicon）で実行しますか？**
- はい → mlx-whisper（GPU・高速）を既定で使う
- いいえ、または不明 → 自動的に faster-whisper（CPU）にフォールバックする（速度は落ちるが動作は変わらない）

**Q6. `faster-whisper` と `mlx-whisper`（Apple Siliconの場合）はインストール済みですか？**
- はい → 設定のみ生成
- いいえ → インストール手順もあわせて出力する

## スキル設定

**Q9. コマンド名は何にしますか？**
- デフォルト: `srt-fast`（`/srt-fast` として呼び出す）

**Q10. Premiere Pro のカット点同期XMLを使う運用にしますか？**
- はい → 使うたびに `--xml` オプションでXMLパスを渡す手順を含める
- いいえ → WAV/動画単体からの生成のみでよい（後から使いたくなったら追記可能）
