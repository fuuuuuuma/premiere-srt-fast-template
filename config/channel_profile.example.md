# チャンネル設定（テンプレート）

これはテンプレートです。実際の設定は `config/channel_profile.md`（gitignore対象）に書きます。
壁打ちセットアップの回答から自動生成されるので、通常は手で編集する必要はありません。
手動で作る場合はこのファイルを `config/channel_profile.md` としてコピーして値を埋めてください。

## テロップ目標

- 平均文字数の目安: 14字前後（初期値。変更したい場合はここに書く）
- 25字超の許容率: 1%未満（`references/srt_runtime_rules.md` の既定値のまま推奨。変更する場合のみ記入）

## 固有名詞・言い間違い辞書

実際の辞書は `config/corrections.local.json` に持つ（このファイルには方針だけメモする）。
チャンネル名も含め、頻出する専門用語・製品名・人名など、Whisperが誤変換しやすいものが
あれば都度追記する。
