# チャンネル設定（テンプレート）

これはテンプレートです。実際の設定は `config/channel_profile.md`（gitignore対象）に書きます。
`/srt-fast` を初めて実行したとき、このファイルが無ければセットアップ対話で自動生成されるので、
通常は手で編集する必要はありません。手動で作る場合はこのファイルを `config/channel_profile.md`
としてコピーして値を埋めてください。

## チャンネル名

（例: サンプルチャンネル）

## テロップ目標

- 平均文字数の目安: 14字前後（Whisperの初期値。変更したい場合はここに書く）
- 25字超の許容率: 1%未満（`references/srt_runtime_rules.md` の既定値のまま推奨。変更する場合のみ記入）

## 半角スペースの好み

- 既定: 話者ポーズ + 日本語/英数字境界のみ挿入（`references/srt_runtime_rules.md` セクション5参照）
- このチャンネルで既定と違う運用にしたい場合はここに書く

## 固有名詞・言い間違い辞書

実際の辞書は `config/corrections.local.json` に持つ（このファイルには方針だけメモする）。
例: 頻出する専門用語・製品名・人名など、Whisperが誤変換しやすいものがあれば都度追記する。
